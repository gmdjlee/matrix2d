Warpage Analysis 시작 가이드

휨(warpage) 측정 데이터를 불러와 두 면 사이의 갭(gap)을 계산하고, 2D·3D 그래프로 확인하는 프로그램입니다. 이 문서만 따라오면 처음 써도 막힘없이 사용할 수 있습니다.

1이게 뭔가요?

두 개의 휜 면 사이 간격을 계산하는 도구입니다.

제품의 위쪽 면(TOP)과 아래쪽 면(BTM)은 완전히 평평하지 않고 휘어 있습니다. 두 면을 마주 붙였을 때 가장 먼저 닿는 지점을 기준(간격 0)으로 잡고, 나머지 지점이 얼마나 벌어져 있는지를 계산한 것이 GAP입니다.

TOP − BTM 차이에서 가장 작은 값을 0으로 맞추면, 각 지점의 벌어진 정도(gap)가 나옵니다.
계산식은 간단합니다: 차이 = TOP − BTM → 그 중 최소값을 빼서 첫 접촉점이 정확히 0.0이 되도록 맞춥니다. 측정이 비어 있는 셀은 계산에서 자동 제외됩니다.

2프로그램 실행

웹 브라우저에서 열리는 로컬 프로그램입니다.

이 프로그램은 내 PC에서만 도는 웹 앱입니다. 실행하면 브라우저 주소창에 http://127.0.0.1:8050 페이지가 열립니다. 인터넷 연결은 필요 없습니다.

프로그램 폴더 열기

프로젝트 폴더 안에서 명령 프롬프트(터미널)를 엽니다.

실행 명령 입력

python run_app.py

// 잠시 후 "Starting Dash app on port 8050" 메시지가 나옵니다.

브라우저에서 열기

Chrome·Edge 등에서 http://127.0.0.1:8050 로 접속합니다. (자동으로 열리지 않으면 직접 주소를 입력하세요.)

종료

터미널에서 Ctrl + C 를 누르면 프로그램이 종료됩니다.

처음이라 데이터가 없다면, 예제 데이터를 만들 수 있습니다: python scripts/make_sample_data.py → demo_data/ 폴더에 TOP/BTM/GAP/OUT 예제가 생성되고, 프로그램이 자동으로 이 경로를 채워줍니다.

3데이터 준비

4가지 폴더와 파일 이름 규칙만 지키면 됩니다.

4개의 폴더

TOP 위쪽 면

위쪽 면의 휨 측정 데이터 파일들.

BTM 아래쪽 면

아래쪽 면의 휨 측정 데이터 파일들.

GAP 기존 갭 결과

이미 계산돼 있는 갭 파일(있으면 불러와서 볼 수 있음). 없어도 됩니다.

OUT 결과 저장

프로그램이 계산한 갭·이미지를 저장하는 위치.

갭을 새로 계산하려면 TOP 과 BTM 두 폴더만 있으면 됩니다. OUT 은 저장 위치라 미리 비어 있어도 괜찮습니다.

파일 형식

항목내용
확장자.dat · .csv · .txt
구조숫자만 들어간 2D 격자. 제목 줄·행 번호 없음
빈 셀 처리빈 칸, nan, 또는 2000 이상의 값 → 자동으로 "측정 없음" 처리

TOP / BTM 파일 이름 규칙

파일 이름에서 샘플 번호, 시간, 온도를 읽어냅니다. 아래 형식을 지켜주세요.

TITLE제목(자유) _ PT0003PT+4자리 샘플번호 _ 01200s5자리 초 + s (25C)온도 + C, 괄호 .txt

예: PANEL_A_PT0003_01200s(25C).txt → 샘플 3번, 1200초, 25℃

제목(TITLE)에는 _ 밑줄이 들어가도 됩니다. 프로그램은 마지막 _PT 를 기준으로 이름을 해석하기 때문입니다.

GAP / OUT 파일 이름 규칙

계산 결과 파일은 아래 형식입니다. GAP·OUT 폴더의 파일도 같은 형식으로 읽습니다.

{prefix}-{H|C}{temp}_TOP{n}-BTM{m}.txt

예: TEST-C25_TOP3-BTM8.txt → 접두어 TEST, 냉각(C) 25℃, TOP 3번 × BTM 8번

기호의미
H / CH = 가열(Heating), C = 냉각(Cooling)
{temp}온도(℃)
TOP{n}-BTM{m}TOP 샘플 n번과 BTM 샘플 m번의 조합

5기본 사용 순서

이 5단계가 전체 흐름입니다.

폴더 경로 지정

사이드바 Folders 에서 각 칸에 경로를 입력하거나 Browse... 버튼으로 폴더를 고릅니다. 갭 계산만 할 거면 TOP·BTM·OUT 세 칸이면 충분합니다.

Scan 눌러 데이터 읽기

진행 막대가 차고, 파일을 검사해서 유효한 것만 불러옵니다. 이름·내용이 규칙에 안 맞는 파일은 조용히 건너뜁니다(전체가 멈추지 않음). 스캔 후 각 탭의 선택 목록이 채워집니다.

데이터 옵션 점검 (필요할 때만)

측정 방향이 뒤집혀 있거나 회전이 필요하면 Data Options 에서 조정합니다. 기본값 그대로도 대부분 동작합니다.

Gap Compute 탭 → Compute All Gaps

모든 TOP × BTM 조합의 갭을 한 번에 계산합니다. 결과 파일이 OUT 폴더에 저장되고, 오른쪽 표에 목록이 나옵니다.

결과 확인

표에서 결과를 고르면 왼쪽에 2D·3D 그래프가 뜹니다. 온도별 평균 갭은 Effective Gap 탭에서 봅니다.

여기까지가 핵심입니다. 아래 6~8번은 각 탭·옵션의 세부 설명이니, 필요할 때 찾아보면 됩니다.

6탭별 상세

메인 화면 위쪽 4개 탭.

📊 2D View — 한 면씩 평면으로 보기

TOP 샘플과 BTM 샘플을 하나씩 골라 나란히 평면 그래프로 확인합니다.

  • TOP sample / BTM sample — 볼 샘플을 선택
  • Temperature (common) — 두 샘플에 공통으로 있는 온도 선택
  • Chart type — Heatmap(색 격자) 또는 Contour(등고선)
  • Save current figures to OUT as PNG — 현재 두 그래프를 이미지로 저장

🧊 3D View — 여러 면을 입체로 겹쳐 보기

TOP·BTM·GAP·OUT 데이터를 여러 개 골라 3D 곡면으로 함께 표시합니다.

  • Filter: sample no. / temperature — 목록을 샘플·온도로 좁히기
  • TOP / BTM / GAP / OUT datasets — 각 종류에서 표시할 데이터 다중 선택
  • z-offset — 선택한 데이터마다 위아래 높이를 띄워 겹침을 분리 (선택 시 자동으로 입력칸 생김)
  • Save current figure to OUT as PNG — 현재 3D 화면 저장

🟣 Gap Compute — 갭 계산 (핵심 탭)

모든 TOP × BTM 조합의 갭을 한꺼번에 계산하고 결과를 저장·확인합니다.

  • Output name prefix — 저장 파일 이름 앞에 붙일 접두어 (예: TEST). 비우면 GAP
  • Compute All Gaps — 전체 조합 계산 시작 (진행 막대 표시, 백그라운드 처리)
  • Save All Images (2D+3D) — 계산된 각 결과를 _2D.png·_3D.png 로 일괄 저장
  • 왼쪽: 결과 하나를 골라 2D·3D 그래프로 확인 / 오른쪽: 결과 목록 표(정렬·필터·페이지 지원)

저장 파일: {prefix}-{H|C}{temp}_TOP{n}-BTM{m}.txt (탭 구분). 같은 이름이 겹치면 _2, _3 이 붙습니다. 전체 요약표 {prefix}.txt 도 함께 저장됩니다.

계산 중 다른 탭으로 이동해도 계산은 계속됩니다. 온도가 ±2℃ 이내면 같은 온도점으로 취급해 짝을 맞춥니다.

📈 Effective Gap — 온도별 평균 갭

마지막 Gap Compute 결과를 바탕으로, 온도점마다 모든 TOP-BTM 조합의 평균 갭을 선그래프로 보여줍니다. 오차 막대는 표준편차입니다. 가열 구간이 오름차순으로, 이어서 냉각 구간이 내림차순으로 배치됩니다.

Save current figure to OUT as PNG 로 저장할 수 있습니다.

9문제 해결

자주 겪는 상황과 해결법.

Scan을 눌렀는데 "데이터 없음" 오류가 나요

폴더 경로가 맞는지, 파일 이름 규칙(3장)과 내용 형식(숫자 격자, 제목/행번호 없음)이 맞는지 확인하세요. 규칙에 안 맞는 파일은 자동으로 제외되어 유효 파일이 0개가 되면 이 메시지가 나옵니다.

파일 목록에 특정 파일이 안 보여요

그 파일이 이름 또는 내용 검사에서 걸러졌을 가능성이 큽니다. 로그 파일에 건너뛴 이유가 기록됩니다: logs/matrix2d.log.

진행 막대가 멈춘 것처럼 보여요

큰 파일이 많으면 시간이 걸립니다. 계산은 백그라운드에서 진행되며, 계산 중 다른 탭으로 이동해도 됩니다. 페이지를 새로고침하면 진행 중이던 작업과 화면 상태가 초기화되니 피하세요.

브라우저 페이지를 새로고침해도 되나요?

가급적 하지 마세요. 새로고침하면 불러온 데이터·계산 결과 상태가 사라집니다. 다시 Scan부터 해야 합니다.

결과 파일은 어디에 저장되나요?

OUT 폴더에 저장됩니다. 갭 파일(.txt), 요약표({prefix}.txt), 이미지(_2D.png·_3D.png)가 이곳에 쌓입니다.
